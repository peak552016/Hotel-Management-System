# Intial Installation
    1. hotel-react-web (Frontend)
        - cd hotel-react-web
        - npm install
    2. hotel-api-nodejs (backend)
        - cd hotel-api-nodejs
        - npm install

# Instructions: 
    1. Download Node.js into your computer (https://nodejs.org/zh-cn/)
    2. Download the program file 
    3. Extract the zip file
    4. Create a file name 'hotel' in C:\xampp\htdocs
    5. Open xampp go to http://localhost/phpmyadmin/ 
    6. Create a database name hotel_database [ import the sql file 'hotel_database.sql' in 'hotel-api-nodejs' folder ]
    7. Open a terminal, go to the file location [ C:\xampp\htdocs\hotel\hotel-api-nodejs ], and enter the following command 
        - cd hotel-api-nodejs
        - node index.js
    8. Open another terminal, go to the file location [C:\xampp\htdocs\hotel\hotel-react-web], and enter the following command 
        - cd hotel-react-web
        - npm start
